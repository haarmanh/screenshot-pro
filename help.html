üîß Troubleshooting

Screenshot not working?

  • Refresh the page and try again
  • Check you're not on a restricted page (chrome://, extensions)
  • Ensure extension permissions are granted

Can't see all buttons?

  • Try making your browser window larger
  • Use keyboard shortcuts as alternatives
  • Check if browser zoom is set too high

Editor not responding?

  • Press Escape to cancel and try again
  • Reload the extension in chrome://extensions/
  • Check browser console for error messages
